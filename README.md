# Git研修用リポジトリ

このリポジトリを使ってGitの基本操作を学びます。

## 演習内容

| 演習 | 内容 |
|------|------|
| 基本操作（中級） | `members/` に自分のファイルを作成して commit → push |
| ブランチ（中級） | ブランチを作成して `rules.md` を追加 |
| コンフリクト解消（上級） | `feature-B` を main にマージして event.md のコンフリクトを解消 |

## 使い方

1. 「Use this template」→「Create a new repository」
2. リポジトリ名を入力して作成
3. VSCodeでclone
4. 講師の指示に従って演習を進める

## ファイル説明

- `members/` - 各自の自己紹介ファイルを作成するフォルダ
- `event.md` - 懇親会の企画書（上級のコンフリクト演習で使用）
- `rules.md` - 懇親会のルール（中級のブランチ演習で作成）
